# DouYinService
某音的加密签名解密、视频Id、视频分享链接获取无水印视频、视频信息的接口分享

##  测试使用
- 获取  `_signature` 解密值 可见 `get_signature.py`源码 。查看 返回数据：
     ```commandline
    python get_signature.py
    ```
- 通过 视频ID 或者分享链接 获取 无水印视频 下载链接 及视频信息 可见 `get_video.py`源码 。查看 返回数据：
    ```commandline
    python get_videos.py
    ```
